# Licencias • Vacaciones (React)

UI moderna estilo “Skeleton” (cards suaves, glass, grid glow) + Timeline visual (±15 días) + vista por mes.

## Requisitos
- Node.js LTS (18/20/22). Recomendado: 20 o 22.
- npm (viene con Node).

## Instalación (Windows CMD)
1) Descomprimí el ZIP en:
`C:\Users\facundo.ardusso\OneDrive - Suizo Argentina S.A\Anterior\Escritorio\Licencias`

2) Abrí CMD y ejecutá:

```bat
cd "C:\Users\facundo.ardusso\OneDrive - Suizo Argentina S.A\Anterior\Escritorio\Licencias\licencias-saas-react"
npm install
npm run dev
```

3) Abrí el navegador en:
http://localhost:5173

## Scripts
- `npm run dev` (desarrollo)
- `npm run build` (build producción)
- `npm run preview` (preview del build)
